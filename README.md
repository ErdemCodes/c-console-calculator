# C Console Calculator

A simple console-based calculator written in C.

This program allows users to perform basic mathematical operations using a menu-driven interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Square Root
- Exponentiation

## Technologies
- C Programming Language
- Standard Libraries: stdio.h, math.h

## How to Compile and Run
```bash
gcc calculator.c -o calculator -lm
./calculator
